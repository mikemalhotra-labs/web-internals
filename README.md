# web-internals
Notes and small experiments to understand basic web internals such as HTML structure, browser behavior, forms, and client-side scripts.

This repository contains small experiments and notes created while learning
how the web works at a basic level.

Topics explored here include:
- HTML document structure
- Form behavior
- Basic client-side scripting
- How browsers interpret markup

Most files are simple and created for learning purposes.
Some experiments are incomplete or outdated.
